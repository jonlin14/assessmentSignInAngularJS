##Description
This application will simulate a sign-in sheet. Students will be able to click on their name to sign in or sign out for the day. The teacher will be able to see who's signed in or signed out on another view.

##Technologies Used
JavaScript
<br>

<br>
HTML
<br>
CSS with Bootstrap
<br>





##Use and Editing
To use the app, download the files from github. Open the index.html file in the browser of your choice. The webpage will open and allow the visitor to use the site.  <br>
